# MYPOFO API

This api developed to serve data public and private endpoints to my portfolio websites.

## Languages & Framework

- NodeJS
- Express
- MongoDB

### How to run

- Make sure .env exists in below format

```
PORT: port_number
MONGODB_URI = "db uri"
NODE_ENV = 'development'

```

- Install dependencies by running below command
  `npm i`

- Run the api
  `npm start `
